This program will produce a multiplication table based on the values you input into the form.
Please enter start and end numbers for both the horizontal axis and vertical axis.
Please enter integers only (positive or negative). If you enter a decimal, everything after the decimal point will be deleted.
Make sure the starting number is less than the ending number for both the horizontal and vertical axes.
The acceptable range of numbers is from -50 to 50.
If you want to add the table to a tab click the green add table to tab button. You can individually delete tabs
by clicking the x in the corner or multi-delete by checking the check boxes of the desired tabs then clicking "Delete All Selected Tabs" button.
